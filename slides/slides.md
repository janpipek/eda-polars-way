## Intro

---

### What is polars?

- Library for manipulation with tabular data
- Contender to pandas (and many more similar tools)
- Since 20XX, started by Ritchie Vink

---

### Why polars?

- Performance
- Clean(er) API 
- Query optimization
- Cool kid on the block

---

### Why not polars?

- Less stable
- Less functionality
- Less known
- Sometimes lengthy code
